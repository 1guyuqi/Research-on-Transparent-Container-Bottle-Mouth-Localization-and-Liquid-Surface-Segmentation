# Research-on-Transparent-Container-Bottle-Mouth-Localization-and-Liquid-Surface-Segmentation
this study focuses on complex robotic arm manipulation scenarios. By combining Vision-Language Models (VLM) and deep learning-based instance segmentation, we perform multi-scenario data acquisition, bottle mouth judgment, liquid surface segmentation model training, and RoboEngine data augmentation.
Using sam_labeler, we manually segment the liquid surface of Erlenmeyer flasks, with a step of visual restoration of annotations. The annotated data is then converted into standard YOLO format for machine learning. The binary mask of the bottle is extracted, and RoboEngineRobotSegmentation is employed to automatically segment the robotic arm. After merging the masks, texture background enhancement is performed. Specifically, the bottle region mask is extracted from the training set, and texture enhancement is applied to other regions. For each original image, three augmented images are generated via background replacement, increasing the dataset to four times its original size, thus constructing a high-quality dataset for model training. Two sets of YOLOv8n-seg.pt training are conducted for the liquid surface segmentation model: the first group uses only the training set, and the second group uses the training set plus the augmented set. The segmentation performance of both groups is evaluated on the test set, and key metrics are compared to verify whether data augmentation brings performance improvement.
